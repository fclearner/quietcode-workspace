# AlgoLab · 本地刷题工作台

一个不依赖第三方 npm 包的 LeetCode 风格本地练习站。题库数据来自相邻目录中的
[`leetcode-company-wise-problems`](https://github.com/liquidslr/leetcode-company-wise-problems) 仓库。

## 启动

需要 Node.js 20+；运行 Python 代码需要 `python3`，运行 C++ 代码需要 `g++`。

```bash
cd leetcode-local-app
npm run import
npm start
```

浏览器打开 <http://127.0.0.1:3210>。需要更换端口时可使用 `PORT=8080 npm start`。

## 已实现

- 导入 470 家公司的 3,257 道去重题目，保留公司、时间区间、频率、难度、通过率和标签。
- 题目搜索、难度/公司/状态/标签筛选、排序、分页、随机题和每日一题。
- 公司题库、练习进度、活动热力图、提交历史、连续练习天数和每日目标。
- 收藏、草稿自动保存、解题笔记、自定义测试用例、深色模式和数据导入/导出。
- JavaScript、Python 3 和 C++17 的标准输入/输出运行环境；内置题支持样例判题。
- 代码执行 5 秒超时、64 KB 输出上限、请求大小和测试数量限制。
- 响应式桌面/移动页面，无数据库、无登录、无云端依赖。
- 自适应学习计划：根据完成状态、失败次数、提交通过率、连续天数和题目标签生成下一项训练。
- 动态训练队列、薄弱专题掌握度、难度升降级、七天间隔复盘和每次提交后的即时建议。
- 初始编辑器只提供输入解析、函数签名和 `TODO` 骨架；通过全部测试后才解锁参考实现。
- 题目页内置 Codex 学习教练，可结合当前代码、测试用例、运行结果和薄弱专题给出逐级提示。
- 三道内置完整题面分别配置 27、35、30 个服务端隐藏用例；提交会同时检查公开、自定义和隐藏用例，隐藏失败时不泄露输入与答案。

## 学习教练

在题目页点击“问教练”，可以直接描述卡点，或使用“我没思路”“分析失败用例”等快捷提问。教练只在你主动发送消息时调用本机的 Codex CLI；未完成题目时默认只给最小有效提示，通过后才允许讨论完整实现。

使用前需要安装并登录 Codex CLI（`codex --version` 可检查）。请求会消耗当前 Codex 账号的用量，并将本轮题目、当前代码、测试用例、最近运行结果和最近十轮对话发送给 Codex 服务。对话记录仍保存在浏览器 `localStorage` 中。

## 学习引导如何调整

“学习计划”会在页面加载以及每次运行、提交后重新评估：

- 尝试过但没有完成的内容会优先回到队列，连续失败会提高同主题巩固权重。
- 新用户优先高频基础内容；稳定完成后逐步从简单调整为中等和综合挑战。
- 正确率较低时降低难度，避免连续堆叠挫败；通过率稳定后增加迁移训练。
- 完成超过七天的内容进入间隔复盘候选。
- 自适应画像只使用本地练习记录；只有主动向学习教练发消息时，相关题目上下文和代码才会发送给 Codex 服务。

## 补充公司题单

主数据之外，导入脚本还会合并 [`data/supplemental-company-lists.json`](data/supplemental-company-lists.json)。当前补充了两个独立 GitHub 来源中主快照缺少的 13 个 ByteDance/TikTok 候选，并在题目详情保留来源和快照日期。补充数据不会覆盖更新主快照中的更高题频。

## 数据说明

上游仓库只有题目元数据，并不提供题面、函数签名、隐藏测试或官方答案。因此：

- `Two Sum`、`Valid Parentheses`、`Best Time to Buy and Sell Stock` 内置了本地中文题面、模板和测试。
- 其他题目可以通过页面中的原题链接阅读题面，并在本地以标准输入/输出模式运行代码、添加用例、保存笔记和记录完成状态。
- 只有上述内置题具备可信的服务端隐藏判题；纯元数据题提交时只能校验你填写的用例，页面会明确标注覆盖范围。
- 浏览器侧进度保存在 `localStorage`；可在“进度”页面导出 JSON 备份。

## 安全边界

服务默认只监听 `127.0.0.1`。Python/C++ 使用独立临时目录和子进程，JavaScript 使用限制模块访问的 Worker；所有执行器均有超时和输出上限，但这仍不是操作系统级沙箱。不要把服务暴露到公网，也不要运行不可信代码。若需要多用户、容器隔离、账号同步、讨论区或完整官方题面，需要另行接入认证、数据库、容器执行器和合法题面数据源。
